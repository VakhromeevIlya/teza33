Типы котельных

Мы проектируем, строим и обслуживаем промышленные котельные различных типов. Специалисты Миг-Плюс подберут наиболее оптимальное решение для системы теплоснабжения, исходя из имеющихся условий и необходимых технических характеристик. Обычно котельные классифицируют по виду топлива, по способу размещения и по типу котлов.
По типу топлива
  • Газ
  • Жидкое топливо
  • Электричество
  • Комбинированный тип
По способу размещения
  • Стационарные
  • Блочно-модульные (БМК)
  • Модульные котельные
По типу теплоносителя
  • Паровые
  • Водогрейные
  • Комбинированные
  • На диатермическом масле
